Title: (1751) OGSA Basic Security Profile 1.0 - core
Description: 
This profile addresses common security issues to all OGSA services. This profile is expected be used with other OGSA 
profiles, e.g. OGSA WSRF profile 1.0 and also be extended by the other OGSA Basic Security Profiles, e.g. secure channel
 (which is submitted simultaneously)..
