ByteIO Interoperability Fiesta

Deadlines

  • Public Endpoint Available for testing on close-of-business Friday, 20 July 2007 where close-of business is defined as 9am GMT on the following Monday

Interoperability Scenario

The interoperability fiesta will be governed by the following documents
  • ByteIO Recomendation Document
  • ByteIO WSRF Rendering Document
  • ByteIO Interoperability Testing Scenarios

Public Endpoints

Note that all public endpoints listed in this section refer to the EPR of a valid ByteIOInterop factory endpoint as defined in Appendix A of the ByteIO Interoperability Testing Scenarios document. Further note that clients MUST be able to utilize both valid URLs (for degenerate EPRs not requiring ResourceParameters or metadata) as endpoints as well as full XML documents (or links thereto) representing a valid XML serialization of a WS-Addressing EndpointReferenceType.
