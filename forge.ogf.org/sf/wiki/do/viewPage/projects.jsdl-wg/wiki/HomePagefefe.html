JSDL Teleconference Infrastructure

JSDL uses Hidefconferencing provided by EGI.eu. The conference room number is 8111664 and is required to be entered when using the phone bridge.

Skype access

You may use Skype to dial into the conference; for this add the following phone number to your Skype contacts: +9900827048111664

This number already contains the conference room number so that you can join the call at a single click.
