Requirements

This section deals with the requirements. All requirements are grouped. Every group is described by a table holding a identifier for the requirement, a short description, and severity of the issue (Which can be low, medium or high).

General Requirements

id Requirement Severity Details
G010 Use a single port/socket connection for simple transfers Medium Jep somehow we need to get through the firewall, I know this is a very "unclear" requirement
G020 Grid resources need to be protected by firewalls High
G030 Detect number off ports needed before, after, and during execution High
G040 The network and application need to deal with security policies High

Requirements on Hardware on the communication path

id Requirement Severity Details
H010 Routers/Firewalls need to be deal with High bandwith Networks High
H020 Number and kinds of Hardware in a routing path need to be known Medium

Requirements for the Support of Grid Middleware Solutions/Protocols

id Requirement Severity Details
M010 Support connection from a ephemeral port on client to a grid service port on server High To support GT/Unicore
M020 Support connection from a controllable ephemeral port on the client to a grid service port on the server High To support GT
M030 Support connection from a controllable ephemeral port on the client to a controllable ephemeral port on the server Medium To support GT
M040 Support the usage of internal and external service EPR Medium
M050 Supoprt the usage of ephemeral internal EPRs which are mapped to external EPRs Medium
M060 Several Audio and Video ports must be supported Medium AccessGrid
