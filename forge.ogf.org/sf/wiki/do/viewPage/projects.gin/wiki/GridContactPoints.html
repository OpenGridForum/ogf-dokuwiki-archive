wiki1595: GridContactPoints
The following grids are participating in GIN. Prospective users should visit the URL listed to learn how to obtain access to each grid's resources. The GIN group does *NOT* negotiate allocation of resources between grids nor act as a broker for negotiating user access. User access process and allocation decisions are independent for each individual grid. This page is intended to serve as a global index of starting points for users' convenience.
  • DEISA
    • Information about obtaining access to the DEISA Grid infrastructure can be found at https://www.deisa.org/userscorner/faq.php#Q2
  • EGEE
    • Information about obtaining access to EGEE Grid can be found at http://egeena4.lal.in2p3.fr/index.php?option=com_content&task=view&id=16&Itemid=32
  • NAREGI
  • UK National Grid Service
    • Information about gaining access to the NGS is detailed at http://www.ngs.ac.uk/access.html
  • NorduGrid
  • OSG
    • contact point for requesting resources on the OSG: goc@opensciencegrid.org
  • PRAGMA
  • TeraGrid
    • Information about obtaining an allocation on TeraGrid can be found at http://www.teragrid.org/userinfo/access/accounts.php
  • GridAustralia (formerly known as the APACGrid)

This page is still being populated and represents a subset of grids participating in GIN.
