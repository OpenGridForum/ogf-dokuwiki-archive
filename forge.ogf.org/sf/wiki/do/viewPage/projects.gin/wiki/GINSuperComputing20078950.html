GINSC2007-002: SRB/SRM island interoperabilities

Plans on GIN-DATA list to achieve interoperability using the SRM interface with SRB.

Area: GIN-DATA

Category: Production

Status: A collaboration is sought with a group that has a Java-based version of the SRM interface. We will then seek a port on top of the Java interface to the SRB, JARGON. Such a port will also provide an SRM interface to the iRODS rule-oriented data system.
