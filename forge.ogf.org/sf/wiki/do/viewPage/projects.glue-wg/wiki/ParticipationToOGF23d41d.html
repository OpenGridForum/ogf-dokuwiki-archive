GLUE-WG - Participation to OGF 23

OGF23Minutes

GLUE 2 Specification - Overview

Session Leaders

Sergio Andreozzi, Laurence Field, Balazs Kònya

Date and Duration

4/5 June 2008

Description

The GLUE-WG aims to provide a recommendation for an abstract information model which is expressed via a schema independent of information system implementations. After one and half year of work, the specification document is in public comment state.

The purpose of this session is to present and discuss the specification of GLUE 2.0.

Preliminary Agenda

  • introduction

Constraints

  • expected participants: 26-50
