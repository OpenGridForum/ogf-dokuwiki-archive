wiki2190: PhoneMeeting20090612
  • Date: 12 June 2009
  • Time: 4:00 PM CET (conversion to other timezones)
  • Duration: 1 - 1,5h
  • Calling number: +41 22 76 77000 (ask to be connected to "GLUE Meeting")
  • Shared screen session information:
    • URL: http://glue-wg.glance.net
    • Key: 1206

Agenda

  • David: Update on schema changes
    • Groups requires locally unique IDs instead of global ones.
    • More basic tests: Minimum attributes per objects and all attributes per object.
  • David: Mirroring of the OGF subversion repository. (http://cern.ch/glue)
  • Laurence: Set up a top level BDII containing the examples.
  • Laurence: will look in the XML example to find out where groupings are used.
  • Laurence: Investigate if the OGF subversion repository can be world readable.
  • David: When to start packaging process?

Attendance:

  • Stephen Burke
  • David Horat
  • Maarten Litmaath
  • Paul Livesey
  • Laurence Field

Minutes

  • An update on the latest changes was given by David. The grouping object was added along with updated tests.
  • Some comments were made on the example LDIF, it still needs more work.
  • A discussion on hosting of components was inconclusive. Maarten highlighted the need to ensure that the hosting was secure.
  • From the XML schema, it seems that groups are used for Service, Share and Activity.
  • The packaging of the schema was considered. Within EGEE, GLUE 2.) will be added to the existing glue-schema rpm.

Actions

  • Laurence will produce an update BDII and glue-schema package by the end of next week. This will be for both the SL4 and SL5 release.
  • Laurence will follow up on the investigation to see if the OGF subversion repository can be world readable.
  • Laurence will discuss with Balazs DIT, Grouping and implementation when he is back from vacation.
