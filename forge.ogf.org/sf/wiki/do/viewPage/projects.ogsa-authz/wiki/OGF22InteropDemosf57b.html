OGSA Authz WG interoperability demonstrations at OGF 22

This page contains information on the demonstration that are going to be given at OGF 22 in Boston to show adoption of OGSA AUthz WG specifications.

Attribute Exchange Profile Demo

The Attribute Exchange Profile defines a profile in which a consumer requests assertions from an Attribute Service that associates a set of attributes with a subject. It is based on the OASIS Security Assertion Markup Language (SAML) V2.0 for requesting and expressing attribute assertions. In particular, it extends the SAML V2.0 Deployment Profile for X.509 Subject that in turns extend the SAML Assertion Query/Request profile, describing how SAML Subjects are used for representing a principal that owns a X.509 identity, and how assertions regarding such a principals are exchanged and consumed. See working draft.

Participants: VOMS, UVOS
