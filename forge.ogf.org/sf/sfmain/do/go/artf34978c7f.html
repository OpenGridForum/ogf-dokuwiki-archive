Title: (198) Use of SAML for OGSA Authorization
Description: 
This document defines an open grid services architecture (OGSA) authorization service based on the use of the security 
assertion markup language (SAML) as a format for requesting and expressing authorization assertions. Defining standard 
formats for these messages allows for pluggability of different authorization systems using SAML. .
