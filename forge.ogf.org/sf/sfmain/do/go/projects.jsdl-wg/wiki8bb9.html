JSDL Teleconference Infrastructure

Audio conferencing

JSDL uses Hidefconferencing provided by EGI.eu. The conference room number is 8111664 and is required to be entered when using the phone bridge.

Skype access

You may use Skype to dial into the conference; for this add the following phone number to your Skype contacts: +9900827048111664

This number already contains the conference room number so that you can join the call at a single click.

Screen-share service:

The JSDL WG uses Glance.net, kindly provided by UVACSE, for screen sharing purposes.

The number of guests viewing the shared screen is limited to 15.

How to connect

  1. Have a Java enabled browser tready to view the screen.
  2. Browse to the following URL: https://uvacse.glance.net (You may want to set a bookmark.)
  3. The screen share session will ask for a session key. It will always be a four-digit number composed of the conference call's day and month in mmdd format, e.g. 1110 for the 10th November.

If you have trouble connecting

  • To check whether you can connect to Glance, visit http://www.glance.net/site/support/canyouglance.asp.
  • You may want to check your company's firewall configuration, or ask your kind system administrator.
  • Further information may be found at http://www.glance.net/site/support/faq.asp.
