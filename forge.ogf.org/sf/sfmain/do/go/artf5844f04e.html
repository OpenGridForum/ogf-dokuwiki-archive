Artifact artf5844 : FileSystem clarification
Tracker: [CLOSED] JSDL v1.0 Errata
Title: FileSystem clarification
Description: 
Add clarification that this is a requirement of the job. Resources that do not have a filesystem present do not match 
the job requirement. There is no requirement that the filesystem be mounted by the system in response to the request.

Proposed text for resolution in the FileSystem element section:

A resource that does not provide a required filesystem does not satisfy the requirements of the job. It is possible that a separate configuration 
mechanism may setup a resource with the re-quired filesystem (e.g., mount a remote filesystem using NFS) but such configuration is out of scope and 
not required by this specification.
