Submitted by Richard Davies, CEO, ElasticHosts

ElasticHosts is pleased to announce that any patent claims held by
ElasticHosts which are necessary to implement the "Open Cloud Computing
Interface" (OCCI) as designed by the OGF Open Cloud Computing Interface
Working Group will be licensed on a royalty-free basis.

ElasticHosts strongly supports the creation of open and royalty-free
standards for cloud computing, which we believe will be essential for the
success of this key technology.
