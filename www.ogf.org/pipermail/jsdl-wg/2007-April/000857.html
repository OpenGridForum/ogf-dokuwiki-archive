This is a final call for the HPC Profile Application extension for
initial submission to the OGF Editor.

A brief summary:

> This document defines an XML schema for another JSDL Application
> sub-element in addition to POSIXApplication. It is intended for use by
> the HPC Profile being defined with the HPC Profile working group.

Please have a look at the last version on gridforge (version 7) and let
us of know of any comments by next **Thursday, April 19.**
  https://forge.gridforum.org/sf/go/doc13863?nav=1

This is a recommendation track document so after it is submitted to the
 OGF Editor it will enter a 60-day public comment period.
