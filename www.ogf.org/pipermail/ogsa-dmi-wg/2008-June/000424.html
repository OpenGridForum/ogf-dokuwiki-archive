please find attached five documents:
a) OGSA-DMI Functional Specification 1.0 draft 60,
     continuing updating typos, small changes to syntax in XML Schema,  
etc.

b) OGSA-DMI Plain Web Services Rendering 1.0 draft 1,
     which contains message definitions and the WSDL port type for  
plain WS

c) dmi-draft-60.xsd
     the XML Schema for OGSA -DMI Fun Spec draft 60

d) dmi-messages-draft-1.xsd
      The first draft of WS messages for plain WS rendering of OGSA-DMI

e) dmi-porttype-draft-1.wsdl
     ditto
