There is a new version of the glue schema tools for storage at

http://vdt.cs.wisc.edu/software/dcache/tools/glue_schema/0.0.3/schematools-0.0.3.tar.gz

This is less than 3K, and only contains the glue schema 1.3 xml example 
file, the xsl transform, and the python script. You don't need anything 
else if you have installed the 4suite XML processing module. The script 
does an xsl transform on the example site info to produce command lines 
for srmcp and globus-url-copy.  To run the xsl transform, unpack the 
tarball, cd to the directory created, and run

./glue13cmd.py

The site name, storage element name, and user role are set as constants 
in glue13cmd.xsl. There is a second set of these in that file which can 
be uncommented for another example.
