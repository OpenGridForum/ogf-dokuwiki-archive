Hi Stephen,

The Computing Resource represents the whole entity so the Cluster and 
the Batch System that manages the Cluster. The SubCluster concept is now 
called the Execution Environment. This represents a homogeneous set of 
machine which have the same specification.  The Share represents the the 
scheduling policy implemented by the batch system.

I think your misunderstanding comes from thinking that the Cluster and 
Hardware is related.  That hardware is a property of the Execution 
Environment and the Cluster is a "summary" which represents the resource.

Laurence
