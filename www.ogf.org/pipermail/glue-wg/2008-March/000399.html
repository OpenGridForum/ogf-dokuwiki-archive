Dear all,

here is the update on GLUE 2.0 definition:

GLUE 2.0 Spec draft no.29
http://forge.ogf.org/sf/docman/do/downloadDocument/projects.glue-wg/docman.root.drafts/doc14639/29

XSD to draft 29
- namespace: http://forge.ogf.org/sf/go/artf6166 (from now on, namespace 
will change every new draft... pay attention)
- revised list of attributes and order of Endpoint (and derivated 
Storage/ComputingEndpoint)
- revised list of enumeration for Service/Endpoint Capability
- all enumeration small letters
- added attributes for advance reservation (NAREGI use case)
- new source repository for XML and XSD:
http://glueman.svn.sourceforge.net/viewvc/glueman/tags/glue-xsd/draft-29/
