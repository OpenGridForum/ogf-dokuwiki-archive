Grid Projects

There are many large-scale projects aimed at building "Grids" for a variety of applications and research communities. This page contains a list of these projects. This list is not meant to be comprehensive, and no doubt there are projects that have completed and new projects that should be added.

If you have a grid-related project that you believe belongs on this list email office@gridforum.org! Please include the project details: name, URL of project page, countries involved, institutions involved, project period, synopsis, and status. If one of the listed projects has completed, please make us aware of that as well.
