Webcasts

EGA User Forum Webcast Featuring Micron
This is a 70 minute EGA webcast replay. Brooklin Gore, Senior Fellow, Advanced Computing presents "Grid Computing in the Enterprise at Micron Technology". The webcast covers the Micron Global Grid Overview, Governance Best Practices, General Grid Applications, and Processing Models including push, pull and portable. Also, Bob Thome EGA reference model working group chair presents the EGA Reference Model and Use Cases v1.5 technical publication. Also, I need the webcast posted to the webcast landing page under the resource library with same description as above.

EGA User Forum Webcast Featuring UBS
This is a 67 minute EGA webcast replay. Eric Kristoff, IT Director leads a UBS team presentation on compute grids at UBS Investment Bank. The presentation covers drivers and scope, compute grid history and approach, current state of grid, grid computing value curve, the EGA Reference Model, and a sample UBS application - virtualizing calculations. Also, Alan Yoder EGA technology working group chair presents the EGA Data and Storage Provisioning Problem Statement and Approach technical publication.

EGA User Forum Webcast
This is a 74 minute EGA webcast replay. Frank Gillett, Principal Analyst from Forrester speaks on the current state of enterprise grid computing including adoption rates and futures uses of the technology. Also, Bob Thome EGA technology work group chair presents the EGA Reference Model and Security Requirements.

Defining the Grid: An Introduction to the Open Grid Services Architecture
In this Webcast Professor Andrew Grimshaw of the University of Virginia discusses the evolution of grid computing and the OGSA vision of its future. He explains how a service-oriented grid fits with other leading-edge enterprise technologies, and provides details of OGSA and its core capabilities, including progress and plans.
