EMI StAR – Definition of a Storage Accounting Record

Added by Redmine Admin over 9 years ago. Updated over 9 years ago.

Status:published Start date:04/08/2013
Priority:Normal Due date:
Assignee:Greg Newby % Done:

0%
Category:-
Target version:-
Document Type:Informational

Description

In this informational document the EU-project European Middleware Initiative (EMI) describes a storage accounting record (StAR), defined to reflect practical, financial and legal requirements of storage location, usage and space and data flow. The defined record might be the base for a standardized schema or an extension of an existing record like the OGF UR and this document is intended as information to be taken as input for incorporating storage resources into the OGF UR.
