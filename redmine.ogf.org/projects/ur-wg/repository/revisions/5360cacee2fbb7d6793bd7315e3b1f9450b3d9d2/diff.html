  1
  
    
This is the UR-2.0 test suite.


  		





  
  2
  
    

  		





  
  3
  
    
It contains:


  		





  
  4
  
    

  		





  
  5
  
    
  o  a XSD for validating XML data (ur-2.0.xsd)


  		





  
  6
  
    
  o  a validation suite for validating XML data  (Makefile, which uses ur-2.0.xsd and xmllint)


  		





  
  7
  
    
  o  several example documents that illustrate the format (ur2-*.xml.gz)


  		





  
  8
  
    

  		





  
  9
  
    
For description of the UR-2.0 format, see http://redmine.ogf.org/dmsf/ur-wg


  		





  
  10
  
    

  		





  
  11
  
    
To use the validation suite, do


  		





  
  12
  
    

  		





  
  13
  
    
  make test      --- check all .xml and .xml.gz files in current directory


  		





  
  14
  
    
  make testclean --- clean up after running the tests


  		





  
  15
  
    
  make clean     --- more aggressive cleaning: removes backup files from editor


  		





  
  16
  
    

  		





  
  17
  
    
Please email comments to Jon Kerr Nilsen <j.k.nilsen@fys.uio.no>
