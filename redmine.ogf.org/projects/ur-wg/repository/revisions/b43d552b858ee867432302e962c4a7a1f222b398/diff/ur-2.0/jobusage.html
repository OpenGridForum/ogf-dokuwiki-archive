\subsection{Charge}


  		





  48
  65
  
    

  		





  49
  
  
    
This is property may be used for economic accounting purpose. It is the charge applied to the user resource usage.


  		





  50
  
  
    
\begin{itemize}


  		





  51
  
  
    
\item When specifying \emph{Charge}, the following two attributes MUST be specified:


  		





  52
  
  
    
\begin{itemize}


  		





  53
  
  
    
\item \emph{unit} – defines the currency used to report the charge as defined in ISO 4217\footnote{http://www.iso.org/iso/currency\_codes}.


  		





  54
  
  
    
\item The \emph{unit} attribute type MUST be an ISO 4217 alphabetic code.


  		





  55
  
  
    
\item \emph{formula} – The cost computation formula adopted to compute the applied charge.


  		





  56
  
  
    
\item The \emph{formula} attribute type MUST be a string.


  		





  57
  
  
    
\end{itemize}


  		





  58
  
  
    
\item The \emph{Charge} field type MUST be a float.


  		





  59
  
  
    
\end{itemize}


  		





  60
  66
  
    

  		





  61
  
  
    
\begin{XMLexample}


  		





  62
  
  
    
<ur:Charge ur:formula="42 units/CPU/day" ur:unit="USD">1.75</ur:Charge>


  		





  63
  
  
    
\end{XMLexample}
