\subsection{Status}


  		





  
  34
  
    

  		





  
  35
  
    

  		





  
  36
  
    
\begin{itemize}


  		





  
  37
  
    
\item This property SHOULD be referred to as status.


  		





  
  38
  
    
\item This property MUST contain data of type string.


  		





  
  39
  
    
\item This property MUST exist.


  		





  
  40
  
    
\item This property MUST support the following values:


  		





  
  41
  
    
  \begin{itemize}


  		





  
  42
  
    
  \item completed – The execution completed.


  		





  
  43
  
    
  \item started – Execution started at the time this usage record was generated.


  		





  
  44
  
    
  \item suspended – Execution was suspended at the time this usage record was generated.


  		





  
  45
  
    
  \end{itemize}


  		





  
  46
  
    
\item This property MAY support other values, as agreed upon within the implementation context.


  		





  
  47
  
    
\end{itemize}

\subsection{SuspendTime}


  		





  
  61
  
    

  		





  
  62
  
    
SuspendTime is a value that count fot the time that record the moment in which the Virtual Machine changed status to ``suspended''.


  		





  
  63
  
    

  		





  
  64
  
    
\begin{itemize}


  		





  
  65
  
    
\item SuspendTime MUST be present when Status is set to ``suspended''.


  		





  
  66
  
    
\item SuspendTime field type MUST be an ISO timestamp.


  		





  
  67
  
    
\item time zone may be specified as Z (UTC) or (+|-)hh:mm. Time zones that are not specified are considered undetermined.


  		





  
  68
  
    
\end{itemize}
