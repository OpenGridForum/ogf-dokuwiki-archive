      61
    
    
      

    		
  

  

  

    
      62
    
    
      
\emph{SuspendTime} is a value that count fot the time that record the moment in which the Virtual Machine changed status to ``suspended''.


    		
  

  

  

    
      63
    
    
      

    		
  

  

  

    
      64
    
    
      
\begin{itemize}


    		
  

  

  

    
      65

      66
    
    
      
\item \emph{SuspendTime} field type MUST be an ISO timestamp.


    		
  

  

  

    
      67
