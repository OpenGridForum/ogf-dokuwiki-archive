\subsection{Introducing the OCCI \hl{Attribute} type}


  		





  
  1124
  
    

  		





  
  1125
  
    
The \hl{Attribute} type formalize how attribute properties are represented in


  		





  
  1126
  
    
the OCCI Core Model. Since all attribute properties are optional no modifications


  		





  
  1127
  
    
are necessary in existing implementations to remain compliant.


  		





  
  1128
  
    

  		





  
  1129
  
    
OCCI Infrastructure \cite{occi:infrastructure} define attribute properties for


  		





  
  1130

  1131

  1132
  
    
same.


  		





  
  1133
  
    

  		





  
  1134

  1135
  
    
``mutable'' attribute properties. Future updates may include support for all optional


  		





  
  1136
  
    
attribute properties.
