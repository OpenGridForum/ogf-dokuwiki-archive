293 293 
   OGF-URN  =  "urn:ogf:" SNID ":" SUBNAMESPACE-SPECIFIC-STRING</pre><p id="rfc.section.2.4.p.4">where &lt;SNID&gt; is a unique subnamespace identifier for the specification, and &lt;SUBNAMESPACE-SPECIFIC-STRING&gt; is a unique identifier within the subnamespace identifier scope.</p><p id="rfc.section.2.4.p.5">&lt;SNID&gt; has the same syntax as a &lt;NID&gt; as defined in <a href="#RFC2141"><cite title="URN Syntax">[RFC2141]</cite></a>:</p><div id="rfc.figure.u.2"></div><pre class="inline">
294 294 
   SNID  =  ( ALPHA / DIGIT )  *31( ALPHA / DIGIT / "-" )</pre><p id="rfc.section.2.4.p.7">ALPHA and DIGIT are defined in <a href="http://tools.ietf.org/html/rfc5234#appendix-B">Appendix B</a> of <a href="#RFC5234"><cite title="Augmented BNF for Syntax Specifications: ABNF">[RFC5234]</cite></a>.</p><p id="rfc.section.2.4.p.8">The Technical Director at OGF (or their successors) assigns subnamespace identifiers (SNID).</p><p id="rfc.section.2.4.p.9">The syntax of &lt;SUBNAMESPACE-SPECIFIC-STRING&gt; is dependent on the &lt;SNID&gt;, and MUST be defined by a <a href="#GFD-SERIES">Grid Forum Document</a> <cite title="GGF Document Series">[GFD-SERIES]</cite>. This document does not pose any additional restrictions to the &lt;SUBNAMESPACE-SPECIFIC-STRING&gt; other than what is defined in the NSS syntax as defined by <a href="#RFC2141"><cite title="URN Syntax">[RFC2141]</cite></a> or its successor:</p><div id="rfc.figure.u.3"></div><pre class="inline">
