Une fois le projet cr&eacute;&eacute;, cet identifiant ne peut pas &ecirc;tre modifi&eacute;.</p></li><li><p><span class="guilabel">Champs personnalis&eacute;s</span>: s&eacute;lectionner les champs personnalis&eacute;s que vous souhaitez utiliser pour les demandes du projet. Les r&eacute;visions du d&eacute;p&ocirc;t apparaissent dans la vue <span class="guilabel">Activit&eacute;</span> du projet.</p><p>Pour associer un d&eacute;p&ocirc;t au projet, cocher l'option <span class="guilabel">D&eacute;p&ocirc;t</span> dans la configuration du projet. Un nom de page principale doit &ecirc;tre identiqu&eacute;. Cette page sera la premi&egrave;re page affich&eacute;e lors de la consultation du wiki.</p></div><div class="section" lang="en"><div class="titlepage"><div><div><h4 class="title"><a name="d0e86"></a>1.2.4.&nbsp;Membres</h4></div></div></div><p>Cett section vous permet de d&eacute;finir les membres du projet ainsi que leurs r&ocirc;les respectifs. Un utilisateur ne peut avoir qu'un r&ocirc;le au sein d'un projet donn&eacute;. Le r&ocirc;le d'un membre d&eacute;termine les permissions dont il b&eacute;n&eacute;ficie sur le projet.</p></div><div class="section" lang="en"><div class="titlepage"><div><div><h4 class="title"><a name="d0e91"></a>1.2.5.&nbsp;Versions</h4></div></div></div><p>Les versions vous permettent de suivre les changements survenus tout au long du projet. La date figurant sur la version correspond &agrave; sa date d'&eacute;ch&eacute;ance. Les versions pour lesquelles la date n'est pas encore pass&eacute;e sont affich&eacute;es sur la vue "Roadmap".</p><p>Vous pouvez associer une page wiki &agrave; chaque version du projet.
