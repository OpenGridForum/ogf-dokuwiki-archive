The basic abstract element of the schema is the \emph{Network Element}. All other basic elements inherit from it.


  		





  10
  
  
    

  		





  11
  
  
    
Every Network Element MUST have an \emph{id} attribute, which MUST be a unique URI.


  		





  
  12
  
    
The basic abstract element of the schema is the \emph{Network Object}.  Other


  		





  
  13
  
    
basic elements inherit from it.  The \emph{Network Object} can have a


  		





  
  14
  
    
\emph{Location}, can be related to other instances via a \emph{Relation} and can

  16
  
    
attribute, which MUST be a unique URI.  These characteristics are inherited by


  		





  
  17
