\subsection{Layer}\label{sub:layer}


  		





  
  176
  
    

  		





  
  177
  
    
The \emph{Layer} object describes a specific network technology over which data can be transported. Ports have a \emph{at layer} relationship to describe which Layer they operate at. Instances of Layer objects should use identifiers as specified in Deliverable 3. % TODO.


  		





  
  178
  
    

  		





  
  179
  
    
The possibility of moving data from one layer is described by the Adaptation service. This service specifies its server and client layer to describe between which two layers the data can be transported.


  		





  
  180
  
    

  		





  
  181
  
    
Configured adaptations from one layer to the other are described using the \emph{adaptation sink} and \emph{adaptation source} objects. The first is a relation between two ports at different layers to describe that data can flow from a server layer port to a client layer port.
