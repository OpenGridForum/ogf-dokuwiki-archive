\label{class:adaptation_service}

  223
  
  
    
An \emph{AdaptationService} describes the capability that data from one or more \emph{Port}s can be embedded in the data encoding of one other \emph{Port}. This is commonly referred to as the embedding of client layer (higher network layer) ports in a server layer (lower network layer) port. The \emph{AdaptationService} describes a multiplexing adaptation function, meaning that different channels (the client layer ports) can be embedded in a single data stream (the server layer port).

For example multiplexing several VLANs over a single trunk port.
